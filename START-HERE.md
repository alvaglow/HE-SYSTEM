WHAT YOU NEED TO DO RIGHT NOW
==============================

Step 1: Go to your Downloads folder
        Find: node-installer.msi
        Double-click it and install Node.js

Step 2: Create a FREE GitHub account
        Go to: https://github.com
        Click Sign Up, fill in your details

Step 3: Upload your files to GitHub
        Option A: Double-click UPLOAD-TO-GITHUB.bat
        Option B: Follow the steps in GITHUB-MANUAL.md

Step 4: Deploy to Vercel (FREE hosting)
        Go to: https://vercel.com
        Sign in with GitHub, click Add New Project
        Select hed-system, click Deploy

Done! Your system is now live.
